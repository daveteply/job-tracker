# 📦 ui-components

A library of reusable React components for the Vireo ecosystem.

## 🏗️ Purpose

This package ensures a consistent user experience by providing high-quality, pre-styled, and accessible components. By centralizing UI logic here, we maintain visual consistency across the entire Vireo application while speeding up frontend development.

## 🛠️ Tech Stack

- **Framework:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [daisyUI](https://daisyui.com/)
- **Documentation:** Built-in accessibility and responsive design patterns.

## 🧪 Testing

Run unit tests for this package:

```bash
npx nx test ui-components
```
