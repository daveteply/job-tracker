# ⚙️ Sync Backend

The high-performance Java backend for JobTracker, responsible for real-time data synchronization between devices.

## 🛠️ Tech Stack

- **Framework:** [Quarkus](https://quarkus.io/) (Java 21)
- **Database:** PostgreSQL
- **Migrations:** Flyway for schema evolution.
- **ORM:** Hibernate with Panache for streamlined data access.

## 🏃 Getting Started

Run the development server (requires Docker for PostgreSQL):

```bash
npx nx dev sync-backend
```
